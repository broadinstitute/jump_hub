# Using JUMP data on Terra

[Terra](https://app.terra.bio) is a cloud-based platform for biomedical research, operated by the Broad Institute and Verily. You can use a Terra workspace to explore JUMP Cell Painting data interactively via Jupyter notebooks in a cloud environment, without needing to set up a local computing environment.

## Overview

The Terra workspace provides notebooks for working with data from the [Cell Painting](https://jump-cellpainting.broadinstitute.org/cell-painting) image datasets generated by the [JUMP-CP Consortium](https://jump-cellpainting.broadinstitute.org/).

All data is hosted on the [Cell Painting Gallery](https://registry.opendata.aws/cellpainting-gallery/) on the Registry of Open Data on AWS. The notebooks operate on the principal dataset of 116k chemical and >15k genetic perturbations (`cpg0016`), split across 12 data-generating centers, using human U2OS osteosarcoma cells.

## Setting up the workspace

### Recommended environment settings

| Setting | Value |
|---------|-------|
| CPU Minimum | 1 |
| Disk Size Minimum | 50 GB |
| Memory Minimum | 3.75 GB |
| Environment | Default (GATK, Python, R) |

### Setup steps

1. Clone the workspace and open the **Notebooks** tab.
2. Create a cloud environment by clicking on the setup notebook and following prompts. If you don't have a runtime environment running, it will take a few minutes to create and start.
3. Download the [Explore JUMP profiles](../../scripts/17_explore_profiles.py) script and convert it to a notebook:

    ```bash
    pip install jupytext
    jupytext --to notebook 17_explore_profiles.py
    ```

4. Upload the resulting `17_explore_profiles.ipynb` to the workspace and run it. The notebook reads batch-corrected profiles directly from S3 via the versioned manifest — no local data setup needed.

## About the data

- Most data components (images, raw CellProfiler output, single-cell profiles, aggregated CellProfiler profiles) are available from 12 sources for the principal dataset. Each source corresponds to a unique data-generating center (except `source_7` and `source_13`, which were from the same center).
- See [jump-cellpainting/datasets](https://github.com/jump-cellpainting/datasets) for full details on available and planned data releases.
